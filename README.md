# lazy-img.vue.js
懒加载的vue指令

# 安装
```bash
npm install lazy-img.vue
```

# 代码
```html
<img v-lazy-src="图片url，这里必须是一个变量。如果想直接使用 http 链接，请用单引号包起来">
```

```javascript
// 在某 js 文件里
import 'lazy-img.vue'

// 其他代码
```

> 特别需要注意的是，测试时，要给图片添加初始宽高。  
> 因为正常情况下：
> + 屏幕内的图片被加载，而屏幕外的不加载 ->  
> + 而如果没有初始宽高，则所有的图片一开始时，都挤在一起 ->
> + 导致所有图片“都在屏幕内”，从而失去意义

> 而生产环境下，一般不存在这种问题
